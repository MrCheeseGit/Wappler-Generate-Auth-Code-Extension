# Generate Auth Code examples

Generic API step fragments. Replace step names and downstream connections with your project.

| File | Demonstrates |
|------|----------------|
| [sms-pin.json](sms-pin.json) | 6-digit numeric PIN for SMS verification |
| [email-token.json](email-token.json) | 32-character mixed-case token for email links |

After adding steps to an API, run once and save so Wappler refreshes step `meta` for App Connect bindings.
